### Queue
Queue는 각 요소에 *우선순위를 부여하는 데이터 구조*의 한 종류다.

Enqueue (추가)                       Dequeue(제거)
------------> [Rear][][][][][Front] ----------->

> 우선순위를 부여한다는 의미는 고정된 삽입/제거 메커니즘을 인터페이스로서 제공한다는 뜻이다.
> 중간 원소를 사용자가 직접 제어할수 없고 오직 push/pop 메서드로만 원소를 추가/제거 할 수 있다.
