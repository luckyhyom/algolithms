### Value Representation (값 표현)
메모리 관점에서 1byte 데이터 "01000001"를 어떤 자료형으로 해석하냐에 따라 'A' 혹은 65가 된다.
```cpp
int: 01000001 -> 65
char: 01000001 -> 'A'
```

bit만으로는 데이터를 해석할 수 없다.

### BOJ-10808
```cpp
char s[101] = {0,}; // 0으로 초기화하지 않으면 메모리에 남아있던 값이 들어가게된다.

// a를 입력하였으나 1 0 0 4 6 ... 출력 됨
```

```cpp
// char를 int로 직접 변환하지 않고, char끼리 비교하면 편하다.
    for (size_t i = 0; s[i] != '\0'; ++i) {
        if (s[i] >= 'a' && s[i] <= 'z') {
            alphabet[s[i] - 'a']++;
        }
    }
```
